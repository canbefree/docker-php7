## 阿里云自动构建过程

### 阿里云添加镜像

- 添加镜像仓库，选择github自动海外构建

### 添加标签

```bash
    git tag -a release-v1.0.0 -m "Update ...."
    git push origin --tags
```

### 删除标签

```bash
# 删除本地
git tag -d release-v1.0.0
#删除远程的
git push origin :refs/tags/release-v1.0.0
```
